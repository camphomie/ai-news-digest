# EU AI Intelligence Digest

Öffentliches GitHub-Pages-Repository für veröffentlichte Daily- und Weekly-Reports des EU AI Intelligence Briefing Agenten.

Primäre Report-URL:

```text
https://ai-news.camphausen.eu/
```

GitHub-Pages-Fallback:

```text
https://camphomie.github.io/ai-news-digest/
```

Dieses Repository darf nur veröffentlichbare Dateien enthalten:

- HTML-Reports
- CSS/JS-Assets
- `reports.json`
- öffentliche Archiv- und Indexseiten

Interne Prompts, Memory-Dateien, Quellenbewertungen und Agentenlogik gehören in das private Repository `ai-news-backend`.
